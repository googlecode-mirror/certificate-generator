A) HTML Structure - top

This theme is a fixed layout with one column. All of the information within the main content area is nested within a div with an class of wrapper. The general template structure is the same throughout the template. Here is the general structure.

		
		
		<div id="NavigationInside">
			<ul>
				<li><a href="#">Desktops</a>
					<ul class="horizontal">
						<li><a href="#">Mac Mini</a>
							<div class="extended">
								<h2>Mac Mini</h2>
								<span>Starting at $799</span>
								<img src="images/macMini.png" alt="Mac Mini"/>
								<p>Introducing the new Mac mini. Redesigned in a very big way. Mac mini is the world's most energy efficient desktop computer.</p>
								<ul class="smallNav">
									<li><a href="#">Design</a></li>
									<li><a href="#">Features</a></li>
									<li><a href="#">Why Mac Mini</a></li>
									<li><a href="#">Performance</a></li>
									<li><a href="#">Tech Specs</a></li>
								</ul>
							</div>
						</li>

This is the structure to create the dropdown with picture, Be sure to close all tags. First you start off with the very top level Navigation which is just a ul, then if you want the second level to be horizontal you create a ul within an li with a class of horizontal.

NOTE: In IE 7 this file will have one issue that is the other navigation bars will appear through the dropdown, this is an issue with the navigations all having the same z-index value if you put anything else underneath with a lower z-index it works correctly.

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

	.Navigation ul li a {
		color: #someColor;
	}

B) CSS File - top

I'm using one simple stylesheet to control the style and behaviour of this theme. That file is called AppleMenu.css The heading names are pretty self explanatory.

If your having problems with the Dropdowns appearing below your content try adjusting the z-index value on #Navigation ul li ul li .extended
